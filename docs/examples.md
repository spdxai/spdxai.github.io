# Examples

Explore practical examples of SPDX AI and Dataset profiles to understand how to
document AI systems and datasets.

## SPDX examples repository

The SPDX community maintains a comprehensive collection of examples
demonstrating various SPDX profiles and use cases, including AI and Dataset
examples.

[View SPDX Examples](https://github.com/spdx/spdx-examples){ .md-button .md-button--primary }

## Contributing examples

We welcome contributions of new examples!
If you have interesting use cases or want to improve existing examples:

1. Fork the [spdx-examples repository](https://github.com/spdx/spdx-examples)
2. Add your example with clear documentation
3. Validate the example with spdx3-validate
4. Submit a pull request
5. Engage with reviewers for feedback
