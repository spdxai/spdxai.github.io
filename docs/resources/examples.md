# Examples

Explore practical examples of SPDX AI and Dataset profiles to understand how to document AI systems and datasets.

## SPDX Examples Repository

The SPDX community maintains a comprehensive collection of examples demonstrating various SPDX profiles and use cases, including AI and Dataset examples.

**[View SPDX Examples :material-arrow-right:](https://github.com/spdx/spdx-examples){ .md-button .md-button--primary }**

## What You'll Find

The examples repository includes:

### AI Profile Examples

- **Machine Learning Models** - Examples of documenting trained ML models with their metadata
- **Neural Network Architectures** - Documentation of various neural network types
- **Model Cards** - Standardized model documentation following SPDX format
- **Training Information** - How to capture training data, hyperparameters, and metrics

### Dataset Profile Examples

- **Dataset Metadata** - Examples of comprehensive dataset documentation
- **Data Provenance** - Tracking data sources and lineage
- **Dataset Splits** - Documenting training, validation, and test splits
- **Data Transformations** - Recording preprocessing and augmentation steps

### Use Cases

- **End-to-End AI Projects** - Complete examples from data to deployment
- **Multi-Model Systems** - Documenting complex AI systems with multiple models
- **Transfer Learning** - Examples showing model fine-tuning documentation
- **Federated Learning** - Distributed AI system documentation

## File Formats

Examples are provided in multiple SPDX formats:

- **JSON** - Machine-readable format for automation
- **YAML** - Human-friendly format for manual editing
- **RDF** - Semantic web format for advanced use cases
- **Tag-Value** - Traditional SPDX format

## Using the Examples

These examples serve as:

1. **Learning Resources** - Understand SPDX AI and Dataset profiles
2. **Templates** - Starting points for your own documentation
3. **Reference Implementations** - Best practices for various scenarios
4. **Validation** - Test cases for SPDX tools and validators

## Contributing Examples

We welcome contributions of new examples! If you have interesting use cases or want to improve existing examples:

1. Fork the [spdx-examples repository](https://github.com/spdx/spdx-examples)
2. Add your example with clear documentation
3. Submit a pull request
4. Engage with reviewers for feedback
