## Tools

Explore tools developed by SPDX projects and members of the SPDX AI Working Group.

* **[SPDX Tools](https://spdx.dev/use/spdx-tools/)**: The official suite of tools available on the SPDX project website.
* **[SPDX GitHub repository](https://github.com/spdx/)**: More tools and utilities from SPDX.
* **[SPDX Online Tools](https://tools.spdx.org/)**: A web-based interface for managing SBOMs and license information.
* **[BOM Tools](https://github.com/rgopikrishnan91/aibom)** (beta): Extracts metadata from Hugging Face, GitHub, and arXiv, utilizing LLMs to automatically populate structured BOM fields.
* **[Pitloom](https://github.com/bact/pitloom)** (beta): Generates SBOMs for Python projects. It extracts metadata directly from GGUF, ONNX, PyTorch, and Safetensors models and includes native support for Hatchling build hooks.
